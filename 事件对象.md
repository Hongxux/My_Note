​**事件信息封装：​**​

```
public void actionPerformed(ActionEvent e) {
    // 获取事件发生时间
    long timestamp = e.getWhen();
    
    // 获取事件源
    Object source = e.getSource();
    
    // 获取命令字符串（如有）
    String command = e.getActionCommand();
    
    // 使用这些信息进行智能处理
    if (source instanceof Timer) {
        System.out.println("Timer event at " + new Date(timestamp));
    }
}
```

​**核心价值：​**​

1. ​**上下文提供**​：使回调方法了解事件详情
    
2. ​**统一接口**​：不同事件源使用相同回调签名
    
3. ​**扩展性**​：可添加新事件属性而不改接口
    
4. ​**事件溯源**​：支持日志和调试
