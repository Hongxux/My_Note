​**语法定义**​：在方法参数中使用省略号（`...`）声明可变参数，如`Object... args`。可变参数必须是方法的最后一个参数，且一个方法只能有一个可变参数。
- **自动装箱支持**​：基本类型可变参数（如`double...`）自动装箱为包装类型数组，但优先使用基本类型数组以避免性能开销。
**底层实现**​：编译器将可变参数转换为数组。例如，`max(double... values)`在调用时被转换为`max(new double[]{3.1, 40.4, -5})`，方法内部可像**数组**一样遍历参数。实际使用等同于数组。

**性能注意**​：频繁调用可变参数方法可能产生数组创建开销，在性能敏感场景需谨慎使用。
