|          | **限流**                     | **降级**                   |
| -------- | -------------------------- | ------------------------ |
| **核心目标** | **控制流量速率**，防止系统因突发流量过载     | **保全核心功能**，在系统资源紧张时弃卒保帅  |
| **关注点**  | 系统入口的**流量强度**（如QPS、并发线程数）  | 系统内部的**服务状态**（如响应时间、错误率） |
| **好比**   | 高速公路的**收费站**，控制单位时间进入车辆的数量 | 城市大停电时，**优先保障医院和消防的供电**  |
| **关系**   | **一种具体的降级手段**              | **更上层的容错思想**，限流是其中一种实现方式 |
